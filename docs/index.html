Threshopt v0.1.0

Threshopt is a Python library for automatic threshold optimization of binary classifiers.

Key features:

  • Optimize decision thresholds based on any metric (e.g., accuracy, F1-score, G-Mean, Youdenâ€™s J)
  • Supports cross-validation for robust threshold tuning
  • Easy integration with scikit-learn compatible models
  • Built-in common metrics and ability to use custom metrics
  • Visualize confusion matrices and prediction score distributions (optional)

Get started:

  pip install threshopt
